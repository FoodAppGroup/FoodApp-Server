## Database Container

Declared in the [docker-compose.yml](docker-compose.yml).

* Host: `localhost`
* Port: `3306`
* Database: `foodapp_database`
* Url: `jdbc:mysql://localhost:3306/foodapp_database`
* Username: `root`
* Password: `password`
